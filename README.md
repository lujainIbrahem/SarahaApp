# Social App Backend

Backend API for a social media application built with Node.js and Express.

## Features
- User Authentication & Authorization (JWT)
- User Registration and Login
- CRUD Operations
- File Upload using Multer
- Cloud Storage Integration
- Error Handling

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

```bash
git clone https://github.com/yourusername/project-name.git
cd project-name
npm install
npm run dev
```

## Environment Variables
Create `.env` file and add:

```env
PORT=
DB_URL=
JWT_SECRET=
CLOUD_NAME=
API_KEY=
API_SECRET=
```

## API Endpoints
- POST /auth/signup
- POST /auth/login
- GET /users
- POST /posts
